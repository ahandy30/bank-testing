# ParaBank Testing

These are tests that test the various screens and operations of https://parabank.parasoft.com/parabank/overview.htm

The tests include:
1. Registering a user
2. Existing user login
3. Opening an account
4. Paying a bill
5. Transfering funds from one account to another

How to run:
Download tests in compressed zip folder from Github
Open directory in a text editor (like Visual Studio Code)
If Playwright is not install you can use npm init playwright@latest to initialize 
Type npx playwright test
